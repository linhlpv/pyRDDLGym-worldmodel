# pyRDDLGym-worldmodel
Transformer world models for pyRDDLGym environments.

## Running the Pong Example

`twm/examples/pong_train` provides an example that collects Pong vector data and trains a world model.

`twm/examples/pong_solve` runs a simple random shooting controller on a trained world model.

`twm/examples/pong_image` shows how to train a transformer on simple visual observaitons (pixels) directly.
